API Objective

The objective for this API is to a server-side component for a "movie" web application.
It will provie the users information about a movies' description, director, and genre, etc...
Users will be able to sign up and update personal information, as well as creating a list of their favorite movies.

Movie API Documentation

Get each Genre's data by Name /movies/genre/:genreName GET None A JSON object holding data about a specific genre: 
                        
            {
                "Name": "Drama",

            }
Get each Director's data by Name /movies/director/:directorName GET None A JSON object holding data about a specific director: 
                        
            {
                "Name": "Chrisopher Nolan",
                "Birth": 1970,
                "Bio": "Best known for his cerebral, often nonlinear, storytelling..."
            }
Allow New Users to Register /users POST A JSON object holding data about the user to add, structured like: 
                        
            {
                "name": "Mary",
                "favoriteMovies":[]
            }
A JSON object holding data about a registered user, including an ID: 
                        
            {
                "name": "Mary",
                "favoriteMovies": [],
                "id": "70b7b3bf-4b29-451b-91d5-581d99072bda"
            }
Allow users to update thier name /users/:username PUT A JSON object holding data about the soon to be updated user's information structured like : 
                        
            {
                "name": "Kim Updated Name",
                "favoriteMovies": [],
            }
A JSON object holding data about the UPDATED information 
                        
            {
                "id": 1,
                "name": "Kim Updated Name",
                "favoriteMovies": []
            }
