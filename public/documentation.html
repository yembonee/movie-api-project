API Objective

The objective for this API is to a server-side component for a "movie" web application.
It will provie the users information about a movies' description, director, and genre, etc...
Users will be able to sign up and update personal information, as well as creating a list of their favorite movies.

Movie API Documentation

                "Birth": 1970,

Allow New Users to Register /users POST A JSON object holding data about the user to add, structured like: 
                        
            {
                "Username": "bobthebuilder24",
                "Password": "password123",
                "Email": "bobobob@gmail.com",
                "Birthday": "07/27/02",
            }
A JSON object holding data about a registered user, including an ID: 
                        
            {
                "_id": "65303bd2eb3745ca2394c9a7",
                "Username": "bobthebuilder24",
                "Password": "password123",
                "Email": "bobobob@gmail.com",
                "Birthday": "2002-07-27T07:00:00.000Z",
                "favoriteMovies": [],
                "__v": 0
            }
Allow users to update thier Info /users/:Username PUT A JSON object holding data about the soon to be updated user's information structured like : 
                        
            {
                "Username": "danthedancer24",
                "Password": "password321",
                "Email": "dantheman24@gmail.com",
                "Birthday": "02/13/1992"
            }
A JSON object holding data about the UPDATED information 
                        
            {
                "_id": "65303bd2eb3745ca2394c9a7",
                "Username": "danthedancer24",
                "Password": "password321",
                "Email": "dantheman24@gmail.com",
                "Birthday": "1992-02-13T00:00:00.000Z",
                "favoriteMovies": [],
                "__v": 0
            }
